ОПИСАНИЕ ПРОЕКТА
═══════════════════════════════════════════════════════════════

Редактор схемы вышивки - это специализированное приложение для 
создания схем вышивки крестом из любых изображений. Программа позволяет 
преобразовать обычное изображение в схему вышивки с автоматическим 
построением сетки, созданием палитры цветов и генерацией готовых файлов 
для вышивания.
